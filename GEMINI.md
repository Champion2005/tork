# tork

"all tools. ever."

A minimalist single-page index for toolsets.

## Project Details
- **FQDN:** https://tork.apatel.xyz
- **Tech Stack:** React (Vite), Tailwind CSS, Lucide Icons.
- **Deployment:** Coolify (Nixpacks).
- **GitHub:** [Champion2005/tork](https://github.com/Champion2005/tork)

## Structure
- `src/App.jsx`: Main application logic and tool list.
- `src/index.css`: Tailwind configuration and global styles.

## Adding New Tools
To add a new tool to the index, update the `tools` array in `src/App.jsx`.
